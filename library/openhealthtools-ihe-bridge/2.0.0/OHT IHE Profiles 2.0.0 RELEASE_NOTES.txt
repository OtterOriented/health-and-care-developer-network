  Release 2.0 of the OHT IHE Profiles Project has four distribution packages - 

  1) IHE Profiles Core Plug-ins containing IHE Profiles ATNA, PDQ, PIX, XUA, XDS Components
  2) The Bridge Installation Package containing configuration files, binaries for RPC-style web services 
  3) The Bridge Installation Package containing configuration files, binaries for document literal style web services
  4) The Bridge Standalone Plugins containing just the Bridge core logic plus the IHE core plug-ins

  Javadoc for Release 2.0 can be found in online at:
  https://www.projects.openhealthtools.org/sf/projects/iheprofiles/javadocs/

  This Release now requires the use of Java JRE 6, Eclipse 3.6.x, EMF 2.6.x and MDHT 1.0
